# HOW TO RUN
# FOR LINUX USERS

Fork the CPECLUB_C by using this link

`git clone https://github.com/Lawrence-The-Cinarian/CPECLUB_C`

On your Linux Terminal, Type this

`cd ~/CPECLUB_C/My-first-Calculator`

`make`
